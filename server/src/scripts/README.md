# Скрипты для тестирования

## create-admin.ts

Скрипт для создания администратора в системе.

### Что делает скрипт:

1. **Создает нового администратора** с правами доступа к админ-панели
2. **Или обновляет роль** существующего пользователя на администратора

### Запуск:

```bash
cd server
pnpm create:admin
```

### Настройка параметров:

По умолчанию создается админ с:
- Email: `admin@supermock.com`
- Пароль: `admin123`
- Имя: `Admin`

Вы можете изменить параметры через переменные окружения:

```bash
ADMIN_EMAIL=your-email@example.com ADMIN_PASSWORD=your-password ADMIN_NAME="Your Name" pnpm create:admin
```

### Данные для входа:

После выполнения скрипта используйте указанные email и пароль для входа в систему.

### Примечания:

- Если пользователь с указанным email уже существует, его роль будет обновлена на ADMIN
- Администратор автоматически получает статус PREMIUM
- Скрипт безопасен для повторного запуска

## test-matching.ts

Скрипт для создания тестовых пользователей и тестирования системы матчинга.

### Что делает скрипт:

1. **Создает 5 тестовых пользователей:**
   - Алиса Иванова (Free) - Дизайнер
   - Боб Петров (Premium) - Разработчик
   - Чарли Сидоров (Free) - Маркетолог
   - Диана Козлова (Premium) - Дизайнер
   - Ева Морозова (Free) - Frontend разработчик

2. **Создает тестовые карточки** для каждого пользователя

3. **Тестирует матчинг:**
   - Создает матчи между пользователями
   - Тестирует подтверждение матчей
   - Проверяет лимиты для Free пользователей (максимум 3 матча)
   - Тестирует превышение лимита

### Запуск:

```bash
cd server
pnpm test:matching
```

### Данные для входа:

Все тестовые пользователи имеют одинаковый пароль: `password123`

- alice@test.com / password123
- bob@test.com / password123
- charlie@test.com / password123
- diana@test.com / password123
- eve@test.com / password123

### Примечания:

- Скрипт проверяет существование пользователей перед созданием, чтобы избежать дубликатов
- Лимит для Free пользователей: 3 матча
- Premium пользователи имеют неограниченное количество матчей
